THE TOOLS WHICH ARE USED IN THIS PROJECT ARE JAVA NETWORKING(SERVER SOCKET, SOCKET), JAVA I/O(BUFFEREDREADER, PRINTWRITER, INPUTSTREAMREADER), JAVA COLLECTIONS(HASHSET, SCANNER), MULTITHREADING(THREAD).
